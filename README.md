# Identificación de bacterias zoonóticas mediante 16S metabarcoding

Este repositorio contiene un flujo de trabajo para el análisis de secuencias 16S rRNA con el fin de identificar bacterias zoonóticas, utilizando datos de referencia del estudio:

**Dueholm, M. S., et al. (2020).**
*Generation of Comprehensive Ecosystem-Specific Reference Databases with Species-Level Resolution by High-Throughput Full-Length 16S rRNA Gene Sequencing and Automated Taxonomy Assignment (AutoTax).*  
mBio, 11(5), e01557-20. https://doi.org/10.1128/mbio.01557-20

## Objetivo
Implementar un pipeline reproducible para:
1. Procesar secuencias 16S rRNA.
2. Asignar taxonomía con resolución a nivel de especie.
3. Identificar posibles bacterias zoonóticas según bases de datos de referencia.

## Estructura del proyecto
- `data/`: datos crudos y procesados.
- `scripts/`: scripts de análisis en Python o Bash.
- `results/`: tablas, figuras y reportes.
- `envs/`: archivos de entorno reproducible.
